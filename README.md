# indian-states

A small library to get all Indian States, UT and Capitals.
The indian-states library exported as a [UMD](https://github.com/umdjs/umd) module.

## Installation

<!-- In a browser:

```html
<script src="https://unpkg.com/browse/indian-states@0.1.1/dist/"></script>
``` -->

Using npm:

```shell
npm i -g npm
$ npm i indian-states
```
